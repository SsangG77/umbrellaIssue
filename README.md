## 프로젝트 소개
- 지금부터 8시간동안 우산 필요 유무에 대해서 위젯으로 알 수 있습니다.
- 지금, 오늘, 일주일 날씨를 확인할 수 있습니다.

</br>

## 1. 기술 스택

- 언어: Swift

- 프레임워크: SwiftUI

- 날씨 조회: WeatherKit 

- 네트워킹: Widget Extension
- 위치 추적 및 조회 : CoreLocation
- 백그라운드 : BackgroundTask

</br>
